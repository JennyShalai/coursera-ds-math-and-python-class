Homework folder contains filse with tasks to solve and their implementation for course "Mathematics and Python (Математика и Python для анализа данных)" which is first course (out of six courses) of Machine Learning and Data Analysis Coursera Specialization by MIPT (Специализация Машинное обучение и анализ данных).
Course is four weeks long:
1. intro to Python
2. linear algebra 
  2.1. texts similarity
  2.2. function approximation
3. optimization in python
  3.1. differentiable function minimization
  3.2. global optimization
  3.3. non-differentiable function minimization
4. Central limit theorem
